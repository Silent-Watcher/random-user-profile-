## Installation

Install my-project with npm

```bash
  npm install
  cd dist
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Silent-Watcher/random-user-profile-
```

Go to the project directory

```bash
  cd random-user-profile-
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
